Source arrays


Source array is a one-dimensional array, which is manually defined or generated by the user. Then from this array there is a scan to a two-dimensional array or a carpet according to the Formula, which is also specified by the user. At the moment, the following functions are presented here for setting the Source array: random, digits of Pi, custom massive, second massive


Source array(Исходный массив) это одномерный массив, который задается пользователем вручную или генерируется. Затем из данного массива происходит развертка в двухмерный массив или ковер по Формуле, которая так же задается пользователем. На данный момент здесь представлены следующие функции для установки Исходного массива: random, digits of Pi, custom massive, second massive
